# Demonstration Video Script Guide

## Video Requirements
- **Duration**: 3 minutes maximum
- **Platform**: YouTube, Vimeo, or Facebook Video
- **Visibility**: Public
- **Quality**: Clear audio and screen recording

## Suggested Script (3 Minutes)

### Introduction (20 seconds)
"Hi! I'm [Name], and this is EduQuest - a gamified learning platform. EduQuest makes education engaging through games, rewards, and AI assistance. This project was developed using Kiro, an AI development assistant."

### Feature Demo 1: Gamification System (40 seconds)
**Show**:
- Sign in to the app
- Navigate to shop
- Show 6 different themes (space, beach, forest, arctic, crystal, Halloween)
- Purchase a theme with points
- Switch themes to show visual changes
- Show power-ups in shop

**Script**: "EduQuest features a complete gamification system. Students earn points by learning and can spend them on visual themes and power-ups. We have 6 unique themes, each with custom backgrounds and color schemes. Power-ups like 'Skip Question' and '50/50' help students when they're stuck."

### Feature Demo 2: Quiz Modes (50 seconds)
**Show**:
- Quick Play menu
- Start Lightning Mode (show timer)
- Answer 2-3 questions quickly
- Show Survival Mode (lives system)
- Briefly show Memory Master or Puzzle Quest

**Script**: "We have 5 different quiz modes to keep learning fun. Lightning Mode tests speed and accuracy with a countdown timer. Survival Mode gives you three lives - lose them all and it's game over. Each mode has unique mechanics to engage different learning styles."

### Feature Demo 3: AI-Based FRQ Grading (40 seconds)
**Show**:
- Navigate to FRQ section
- Show an AP Computer Science FRQ question
- Type a free response answer
- Submit and show AI grading in progress
- Display score, rubric breakdown, and detailed feedback

**Script**: "Here's our standout feature - AI-based FRQ grading. Students can answer free response questions like those on AP exams, and our AI instantly grades them using natural language processing. It provides a score, rubric breakdown, and detailed feedback on what was correct and what needs improvement. This is perfect for AP Computer Science exam prep."

### Feature Demo 4: Study Sets & AI Chat (20 seconds)
**Show**:
- Browse study sets screen
- Show premade sets (AP CS, SAT)
- Briefly show AI chat assistant

**Script**: "Students can create custom study sets or use our premade content for AP Computer Science and SAT prep. We also have an AI-powered chat assistant for additional help."

### Kiro Development (30 seconds)
**Show**:
- Open `.kiro/` directory in file explorer
- Show specs folder structure
- Open KIRO_USAGE_WRITEUP.md briefly
- Show hooks folder

**Script**: "This entire project was built using Kiro. We used spec-driven development for complex features, automated hooks for code quality, and steering documents for consistent patterns. Kiro generated over 11,500 lines of code and saved us 60+ hours of development time. All our Kiro documentation is in the .kiro directory."

### Conclusion (10 seconds)
**Show**:
- Return to home screen
- Show theme in action

**Script**: "EduQuest proves that learning can be fun and engaging. Thanks for watching, and check out our repository for the full source code and Kiro documentation!"

## Recording Tips

### Screen Recording
- Use OBS Studio (free) or built-in screen recorder
- Record at 1080p resolution
- 30 FPS is sufficient
- Show cursor for clarity

### Audio
- Use a decent microphone (even phone earbuds work)
- Record in a quiet room
- Speak clearly and at moderate pace
- Consider adding background music (low volume)

### Editing
- Use DaVinci Resolve (free) or similar
- Add text overlays for key points
- Trim any mistakes or long pauses
- Add transitions between sections
- Export at 1080p, H.264 codec
